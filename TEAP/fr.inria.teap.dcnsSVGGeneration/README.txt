This section allows to transform an EXCEL model in a
model conforms to the OMG (omg.org/) standard : TOGAF,
represented in an SVG format.
This was done for the TEAP project (teap-project.org)


******************************************************

Objectives :
-----------

- Transformation of an EXCEL model to another conforms
to TOGAF, in an SVG format.

Tools used :
-----------

- Inkscape
- SVGEdit
- MoDisco
- ATL
- Metrics Visualization Builder (Only some parts)
- XML editor
... And others from the Eclispe Modeling Framework (E.M.F.)
